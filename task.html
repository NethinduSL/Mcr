Task 01 - 10 Marks (6×10=60 for team)

Create a Buzzer worked by ultrasonic senser

Use this code and create a working circuit using TinkerCAD or at home, and bring it or send it. If the team completes it, points will be reserved.
Read the code and understand the ports.

Code↧

long duration; // හඬ තරංග ගමනේ කාලය
int distance; // දුර
int safetyDistance = 10; // ආරක්ෂාව සඳහා දුර (සෙන්ටිමීටර් 10)

void setup() {
pinMode(13, OUTPUT); // පින් 13 (LED) OUTPUT ලෙස සකසන්න
pinMode(9, OUTPUT); // පින් 9 (Ultrasonic sensor - trigPin) OUTPUT ලෙස සකසන්න
pinMode(10, INPUT); // පින් 10 (Ultrasonic sensor - echoPin) INPUT ලෙස සකසන්න
Serial.begin(9600); // Serial Communication ආරම්භ කිරීම (9600 baud rate)
}

void loop() {
// trigPin එක පවා (LOW) තත්වයට දමනවා
digitalWrite(9, LOW);
delayMicroseconds(2); // මතක තබා ගන්නවා
// trigPin එක HIGH තත්වයට දමනවා (10 microseconds)
digitalWrite(9, HIGH);
delayMicroseconds(10);
digitalWrite(9, LOW); // trigPin එක නැවත LOW

// echoPin එකෙන් pulse එකේ කාලය පරික්ෂා කිරීම
duration = pulseIn(10, HIGH); // echoPin එකේ pulse එකක් මැනීම

// distance ගණනය කිරීම (සෙන්ටිමීටර් වලින්)
distance = duration * 0.034 / 2; // sound speed 0.034 cm/μs, distance = duration * speed / 2

// distance <= safetyDistance නම් LED ON කරන්න
if (distance <= safetyDistance) {
digitalWrite(13, HIGH); // LED (or buzzer) පාලනය කරන්න (සක්‍රීය කිරීම)
} else {
digitalWrite(13, LOW); // LED (or buzzer) අක්‍රිය කිරීම
}
// Distance ප්‍රින්ට් කිරීම (Serial Monitor එකට)
Serial.print("Distance: ");
Serial.println(distance); // distance value එක Serial Monitor එකේ ප්‍රින්ට් කරන්න
delay(500); // 500 ms පමණ නතර කරනවා, පසුව නැවත loop එකක් ක්‍රියාත්මක වේ }
